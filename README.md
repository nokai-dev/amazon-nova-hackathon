# Amazon Nova AI Hackathon Project

**Deadline:** March 16, 2026 @ 5:00 PM PDT (Approximately 2 days left)
**URL:** https://amazon-nova.devpost.com/

## Hackathon Details

### Prize Pool
- **Total:** $40,000 cash + $55,000 AWS credits
- First Prize: $15,000 cash + $5,000 AWS credits
- Second Place: $7,000 cash + $5,000 AWS credits
- Category winners: $3,000 cash + $5,000 AWS credits

### Categories
1. **Agentic AI** - Build solutions where agents use Amazon Nova reasoning capabilities
2. **Multimodal Understanding** - Rich multimodal applications (text, docs, speech, images, video)
3. **UI Automation** - Nova Act-powered agents for automating web workflows
4. **Voice AI** - Real-time conversational voice experiences using Nova 2 Sonic
5. **Freestyle** - Other innovative uses

### Requirements
- Must use Amazon Nova foundation models (Nova 2 Lite, Nova 2 Sonic, Nova Act, or multimodal embeddings)
- Must be built on AWS
- Demo video (~3 minutes) required
- Code repo must be shared
- Optional: Blog post on builder.aws.com

### Tech Stack Options
- Amazon Nova 2 Lite (fast, cost-effective reasoning)
- Amazon Nova 2 Sonic (speech-to-speech)
- Amazon Nova Act (UI automation)
- Nova multimodal embeddings
- AWS services (Bedrock, Lambda, S3, etc.)

### Judging Criteria
- **Technical Implementation:** 60% (quality, integration with Nova, architecture)
- **Enterprise/Community Impact:** 20% (business value, community benefits)
- **Creativity and Innovation:** 20% (novelty, innovative use of multi-agent systems)
